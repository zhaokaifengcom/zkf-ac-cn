大家好，欢迎来到赵凯峰的个人网站，在这个站点中，我将更新一些个人日常生活。当然，由于我还运营着一个考研数学网站，该网站也是我生活中重要的一部分，所以，大家在我的个人网站中，也可以看到一些关于「荒原之梦考研数学」网的内容。

当前这个 Personal Website 创建于 2024 年 03 月 29 日，创建这个站点的原因是突然意识到，虽然我在 2017 年就创办了荒原之梦网，但是网站的内容几乎都是知识性的文章，很少有关于我个人的记录。那么，为什么我要专门建设一个个人网站，而不是注册一个社交平台账号呢？这是由于我认为所有的社交平台都是很喧嚣的，而这样一个个人网站只是为了作为我日常生活的记录，没有必要参与这份喧嚣。

正如您所见，当前这个网站几乎没有什么宣传推广，所以，首先感谢能来到这里的读者朋友们。

此外，除了我个人的付出，这个网站目前唯一的成本就是 zkf.ac.cn 这个域名的注册费用——这个网站是一个静态网站，部署在 GitHub 和 Cloudflare Pages, 以目前以及未来一段时间的访问量来看，我几乎不用在服务器上付出成本，而且也不担心各种对服务器的渗透攻击或者拒绝服务攻击。所以，总的来说，相比于需要更复杂运营的「荒原之梦考研数学」网，当前这个小网站，可以让我更省心的写一些有关我个人的内容。

当前这些文章一般是使用 VIM 编辑器完成的，之所以选择使用这个编辑器，是因为 VIM 编辑器的简洁无干扰，与我建立这个个人网站的初衷十分契合，通过 VIM 编辑器在这样一个犹如“世外桃源”的网站上写一些东西，可以让我更加自如的遵循内心的想法。

所以，从某种意义上说，这个网站中的大部分内容都是我的个人日记，但是，大家也可以看到，我事实上使用的是“日志”，因为我觉得，“日志”意味着更有条理，有更多基于事实的记录。

当然，除了作为个人的日志存在，这个网站或许还会开发出更多的内容，比如说，我可能会尝试在这里写小说、发布摄影作品等。但是，基于网站的分工，所有和数学相关的内容，都会发布到 zhaokaifeng.com 而不是这里，所有和考研资讯相关的内容都会发布到 kaoyan.ac.cn 而不是这里。
